Research Statement

I am a PhD student in Computer Science at the University of Illinois Chicago, advised by Prof. Saeid Tizpaz-Niari. My research is centered on Safe Code Generation within Responsible Machine Learning.

My work lies at the intersection of Programming Languages (PL), ML Safety, and Formal Methods. I focus on ensuring the correctness, reliability, and trustworthiness of AI-generated software artifacts by applying rigorous verification techniques and type system design.

Research Areas

Safe Code Generation

Developing techniques to provide correctness and safety guarantees for code produced by large language models. We utilize formal methods to verify that generated artifacts adhere to specified constraints.

[PLDI, POPL, OOPSLA]

Responsible Machine Learning

Investigating the robustness and trustworthiness of ML-driven systems. This includes studying the reliability of AI assistants in critical domains like software development and data analysis.

[ICSE, ASE, FSE]

Programming Language Theory

Applying formal semantics, type systems, and environment models to improve the verification of modern software systems. My work involves implementing interpreters for functional languages like Lettuce.

[TOPLAS, JFP]
